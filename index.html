Om meg

Jeg er Bilal og Jeg er

Jeg er en 21 år gammel dataingeniør student, passionert av progremmering og problem løsning. I universitet så liker jeg å hjelpe andre med å løse utfordringer som de får når de løser algoritmer oppgaver. I min fritid så liker jeg å sitte foran pc og lære meg nye teknologier, og nye tendenser. Akkurat nå så lærer jeg om utvikle raske og komplekse webløsninger med bruk av React på frontend og .Net Core i backend. Jeg har utviklet komplekse applikasjoner som kjøres på datamaskinen med bruk av ren Java på backend. Jeg føler meg komfortabel med bruk av javascript og javascrpit biblioteker på frontend, og med objekt orientert språk som Java eller C# på Backend. Jeg foretrekker å jobbe på agile miljøer, og er en fan av DeVops kulturen. Akkurat nå så lærer jeg om sky teknologier og har planer om å sertifisere meg i Kubernetes i nær framtid. Jeg ofte beskrives som en verdifull ressurs i mine nåværende jobber, og som en dyktig og motivert person. Jeg kan være en veldig verdifull ressurs i en bedrift og til mine kollegaer om kort tid, og er flink til å tilpasse meg inn i nye miljøer." Lære og lære og bare lære videre ..."
